# Pothole Detection Using YOLO

## Overview
This project uses YOLO and OpenCV to detect potholes in road videos. The system processes video frames, identifies potholes using a trained YOLO model, and generates an output video with bounding boxes around detected potholes.

## Features
- Automatic pothole detection
- Video processing
- YOLO object detection
- Bounding box visualization
- Output video generation

## Technologies Used
- Python
- YOLO
- OpenCV
- Ultralytics

## Installation
```bash
git clone https://github.com/muhammadahmadcheema2004/pothole-detection-using-yolo.git

cd pothole-detection-using-yolo

pip install -r requirements.txt
```

## Run

```bash
python source_code/pothole_detection.py
```

## Input
Road video file

## Output
Video with detected potholes highlighted using bounding boxes.
